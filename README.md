## KavNews

Агрегатор новостей на NodeJS (парсер + API) и ReactJS (клиентская часть).

#### Используемые технологии:
- **Frontend:** ReactJS, Redux, Redux Thunk, Recompose, Semantic UI React
- **Backend:** NodeJS, Knex, Cheerio, Unirest
- ECMAScript 6
- Async / Await
